Домашняя работа(Вариант 31)
Парсер для учебного конфигурационного языка с поддержкой констант, массивов, словарей и вложенных структур.

Описание языка
- Числа: целые числа (например: 123, -456)
- Массивы: << значение, значение, ... >>
- Словари: { ключ = значение, ... }
- Константы: def имя := значение
- Подстановки: ?[имя_константы]

Запуск
python3 main.py --input input.txt --output output.toml

Тестирование
python -m pytest tests.py
